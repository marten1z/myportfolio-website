Uber Clone Mobile App

Originally developed as a Java-based simulation, this project has grown into a fully functional Uber Clone Mobile Application built with React Native, Redux, Tailwind CSS, and Google Maps APIs. The app enables users to request rides, view real-time driver locations, calculate dynamic fares, and navigate routes on an interactive map interface.

This project deepened my expertise in cross-platform mobile development, API integration, and state management. I focused on crafting a clean, responsive user interface and implementing efficient data synchronization between front-end components, resulting in a seamless and engaging user experience.

Mock Youtube Homepage

Here I built the famous Youtube home page we all know from scratch using HTML & CSS, from the header to the sidebar, and finally, to each video container. This project boosted both my front-end and back-end skills in terms of web development.

Pier Membership System

This project entails a membership system for 'The Friends of Seaview' pier. It is a simple CLI program that could perform different functions like registering new members, output/display members data & sponsorship messages as well as validate dates.
